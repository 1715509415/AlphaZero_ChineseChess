# harmless

A chinese chess engine

## INSTALL

### GNU/Linux and Mac OS X User

> require

* `python-2.7.x`: <http://python.org>
* `pygame-1.9.x`: <http://pygame.org>


> run

Play with AI
```bash
cd pycchess
python cchess.py
```
* `space`: new game




Replay the latest self-play of training
```bash
cd pycchess
python cchess.py -r
```
